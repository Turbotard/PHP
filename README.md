# PHP
Projet PHP compte bancaire normalement 


Il faut créer un fichier "config.php" dans le dossier src à la racine du dossier, celui-ci doit contenir :

<?php

$config = [
    'db' => [
        'host' => 'localhost',
        'name' => 'bnparihaut',
        'port' => 8888, //avec le bon port suivant votre config de mamp
        'user' => 'root',
        'pass' => 'root'
    ]
]
